# Variabler som kan användas: myname, sender, channel, message
# De skrivs med {} runt då dom byts ut med string formatting.
# Lösa {} skrivs som dubbla {{}}!
# {arg} är speciell: i cmd byts den ut mot (.+?) och i answer byts den
# ut mot vad arg representerade i cmd.
# cmd och answer skiljs med [whitespace]->[whitespace].
# Iom det försvinner trailing whitespace i cmd (men inte leading?)

# cmd är regexes!!!!

# Börja svaret med #py: och det exekveras som python kod istället.
# Observera att variablerna då skrivs utan {} runt!
